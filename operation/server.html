# 众圈小程序平台的特点

1、一次开发，多端运行，一次上线，多端同步上线；

2、全域流量开放给小程序运营方获客；

3、开放、无竞业限制，我们只拒绝非法与低俗的服务与内容。

为了您更好的使用众圈小程序平台服务（以下简称“本服务”），您应当阅读并遵守《众圈小程序平台服务协议》 （以下简称“本协议”）以及与此相关的专项规则等。

本协议中的“两码事公司”指的是珠海两码事信息技术有限公司。

# 1、协议的范围

1.1 本协议是您与两码事公司即本协议的乙方之间关于您使用众圈小程序平台服务所订立的协议。“开发者”是本协议的甲方，指注册、开发、运营、维护小程序的个人或组织，在本协议中亦称为“您”。“用户”是指接受小程序开发者提供服务的用户。

1.2 众圈小程序平台服务是指两码事公司根据本协议向开发者提供的包括远程接口调用、开发框架、服务连接渠道、数据加密传输以及与此相关的一系列互联网技术的服务。您在注册后可以成为众圈小程序平台的开发者，并通过众圈小程序平台创建、开发并发布小程序应用，为用户提供各类服务。

# 2、众圈小程序的开通与审核

2.1 您在完成准入认证并填写相关信息后，您可以对所注册的小程序进行设计开发、内部测试、编辑维护等初始化开发管理。您的小程序发布审核通过前，将无法进行发布，即不会向其他用户进行展示，其他用户也无法搜索、添加该小程序。您应妥善保管您的账户名、密码及密钥，不应随意泄露给任何第三方，由此产生的责任由您独自承担。

2.2 为确保众圈小程序平台、用户等各方的安全、稳定及良好的用户体验，两码事公司将对需要发布的小程序进行发布审核。如您的小程序中存在帐号关系或付费内容，需提供测试号，包含帐号和密码，保证审核者可以体验所有功能。

“发布审核”是指由开发者发起，将其完成初始化开发的小程序提交至两码事公司，由两码事公司自行或委托第三方对该小程序的可操作性、稳定性、用户体验等，进行审查的过程。发布审核结果包括审核通过与审核不通过两种。审核不通过的，该小程序将无法发布。

2.3 您清楚并同意，众圈小程序发布审核服务是两码事公司基于开发者提交包括但不限于注册信息、服务类目、使用说明等资料，在两码事公司的合法权限和能力范围内所综合作出合理、谨慎的形式评估和判断服务。但两码事公司无法实质审查用户的实际经营、运营以及推广等行为，并不为此提供任何担保；也不应因其审核行为而被认定为是小程序的共同提供方，且不因该审核而对开发者的小程序承担任何责任。您应仔细阅读有关众圈小程序平台发布的各项协议、规范和规则，由于客观存在不断变化的法律法规、监管要求、产品规范和用户体验需求等各类因素，您的小程序有可能被拒绝发布。

“拒绝”是指提交发布审核的小程序被审核不通过，或者已经审核通过的小程序，被下线或被拒绝提供本平台服务。

2.4 发布审核通过后，用户可以合理的方式、频率向两码事公司提交版本优化、更新等发布审核申请。

2.5 您理解并同意，发布审核将对两码事公司产生人力和经济负担，若您在审核服务期内，恶意或非恶意（如技术水平、用户需求等）原因而导致过于频繁提交发布审核申请，两码事公司有权对您采取要求支付发布审核服务费、暂停或终止向您提供发布审核服务等措施，由此产生的责任由您独自承担。

2.6 开发者应当如实填写和提交帐号注册与认证资料，完成信息登记，并对资料的真实性、合法性、准确性和有效性承担责任。两码事公司依据您填写和提交的帐号资料，在法律允许的范围内向其他用户展示您的注册信息。如开发者提供的服务或内容需要取得相关法律法规规定的许可或进行备案的，您应当在帐号注册与认证时进行明确说明并提交相应的合法有效的许可或备案证明。否则，两码事公司有权拒绝或终止向您提供本平台服务，并依照本协议对违规帐号进行处罚。因此给两码事公司或第三方造成损害的，您应当依法予以赔偿。

上述资料如有任何变更，开发者应立即向两码事公司书面提交变更后资料。变更资料未经核实前，两码事公司可完全依赖变更前的资料行事，由此产生的一切风险由开发者自行承担。如果两码事公司自行发现，或第三方通知发现开发者提供的上述信息不真实或不准确的，两码事公司有权终止向开发者提供平台服务，由此产生的一切法律责任由开发者自行承担。

2.7 您清楚理解并同意，您发布的小程序都应当在两码事公司所公布的类目范围内。为此，您应当真实、全面、不遗漏地根据您的小程序所提供的服务、从事的活动及所在的行业准确选择类目，使两码事公司更好地识别您所提交的小程序可能涉及的法律法规及监管要求，进行相应的审核。若两码事公司发现您所提交的小程序没有对应的类目，或者所提交的资质材料不符合相关类目的资质材料要求，将不能通过发布审核。同时，基于特定行业、特定领域的法律法规或监管要求，两码事公司也可能根据不断变化的现实情况要求您补充提供相关资质材料，对此您应配合提供，否则两码事公司有权拒绝向您提供平台服务。

# 3、众圈小程序的发布

3.1 审核通过后，您可以在众圈小程序开放平台上发布您的小程序应用，并进行运营/推广、客户服务、维护等各项工作。

3.2 两码事公司仅提供中立的平台服务，两码事公司及众圈小程序平台不参与您的小程序的研发、运营等任何活动。您应自行负责您小程序的创作、设计、信息编辑/展示、加工、修改、测试、运营/推广、客户服务及维护等，并自行承担相应的费用。

3.3 您应保证已发布小程序的使用流畅性、稳定性和安全性，若因小程序自身原因导致其向用户提供的服务中断且未能在两码事公司认为的合理期限内修复的，两码事公司有权采取包括但不限于限制功能直至下线处理，以维护良好的用户体验。

3.4 您理解并同意，两码事公司一直致力于为用户提供文明健康、规范有序的网络环境，您不得利用小程序帐号或众圈小程序平台服务制作、复制、发布、传播违反法律法规的规定以及侵犯用户或第三方合法权益的内容，您不得为其他任何非法目的而使用众圈小程序平台的服务；您不得进行任何违法或不正当的活动，包括但不限于：

（1） 反对宪法所确定的基本原则的；

（2） 危害国家安全，泄露国家秘密，颠覆国家政权，破坏国家统一的；

（3） 损害国家荣誉和利益的；

（4） 煽动民族仇恨、民族歧视、破坏民族团结的；

（5） 破坏国家宗教政策，宣扬邪教和封建迷信的；

（6） 散布谣言，扰乱社会秩序，破坏社会稳定的；

（7） 散布淫秽、色情、赌博、暴力、凶杀、恐怖或者教唆犯罪的；

（8） 侮辱或者诽谤他人，侵害他人合法权利的；

（9） 违反中国法律、法规、规章、条例以及任何具有法律效力之规范的。

3.5 如存在下列情况，两码事公司有权进行警告、限制功能、下线您的某个或全部小程序、拒绝再向您提供任何服务等，并追究您的法律责任，由此发生的用户和开发者之间、或其他任何纠纷应由开发者自行协商解决，两码事公司不承担任何责任。两码事公司有权通过运营规范、规则要求、通知等形式进一步补充或说明本条所规定的内容：

（1） 开发者违反国家相关法律法规定的规定，包括但不限于从事恐怖活动、危害网络安全、洗钱、颠覆政府、煽动民族仇恨等；

（2） 开发者违反本协议约定，或平台运营规范，或两码事公司各项规则要求等；

（3）两码事公司收到司法机关或政府机关或监管机构的相关要求；

（4） 两码事公司收到其他第三方通知，投诉某个开发者或其小程序存在违反法律规定、侵犯他人合法权益等情形；

（5） 两码事公司认为开发者的行为或其小程序可能影响众圈小程序平台的正常运营的，如诱导分享、欺诈舞弊等；

（6） 两码事公司认为开发者存在违反法律规定或不当的行为，包括但不限于未经许可获取、存储、披露用户信息、可能危及网络或用户安全等情形；

3.6 开发者仅得按本协议约定自行使用众圈小程序平台上的权限和接口等，不得为任何第三方申请接口或本协议项下的其他服务。

# 4、用户个人信息保护

4.1 您在申请本平台服务过程中，需要填写一些必要的信息，请保持这些信息的真实、准确、合法、有效并注意及时更新，以便两码事公司向您提供及时有效的帮助，或更好地为您提供服务。根据相关法律法规和政策，请您填写真实的身份信息。若您填写的信息不完整或不准确，则可能无法使用本平台服务或在使用过程中受到限制。您理解并同意，为了向您提供服务，需要在法律允许的范围内向其他用户展示您的注册信息。

4.2 您在运营小程序的过程中，应严格遵守所适用的关于保护用户信息、个人信息和隐私权及保障网络安全运行等方面的法律法规。您的产品、服务具有收集个人信息（包括个人信息及个人敏感信息，其中“个人敏感信息”例如：反映个人财产状况、个人健康生理、个人身份、个人生物识别、网络身份标识、个人电话号码、通信记录和内容、位置状况的数据等以及未成年人个人信息）功能的，应当遵循合法、正当、必要的原则，向用户明示收集、使用信息的目的、方式和范围，并经用户明示同意。

4.3 您保证对从众圈小程序平台获取的、经过用户授权的个人信息进行加密存储并在用户授权范围内使用个人信息，不得在未经用户明示同意或超出用户同意范围的情况下使用、共享、转让和披露用户个人信息；您确保能够建立足够的数据安全能力并采取必要的管理和技术手段，按照国家标准实行个人信息安全保护措施，防止个人信息的泄露、损毁和丢失。同时，未经两码事公司允许或在超出两码事公司授权范围的情况下，您不得收集、处理、存储、抓取或以任何方式获取两码事公司控制的用户个人信息。

4.4 如您违反上述用户个人信息保护条款或两码事公司判断您收集、处理、使用、共享个人信息的行为可能损害个人信息主体权益时，您应按照两码事公司的要求立即删除相关的个人信息数据，并不再收集、处理、存储、使用上述个人信息数据。

业主，是指两码事公司服务运营或两码事公司合作伙伴运营的APP，众圈小程序平台可以运行在该APP上。

当您选择在业主上架时，您需要符合该个业主的相关规则。当您选择停止在某个业主或某几个业主展示时，需要根据业主的相关规则为该业主的用户提供完成已成交的服务或产品等方可停止。

7.5 您同意，任何由于电子邮箱、手机号码、通讯地址等信息错误或其他不可归咎于两码事公司的原因，导致您未收到相关规则、通知、提示等信息的，均不影响该等信息对您所产生的法律效力，并您应受其约束，由此产生的一切后果及责任由您自行承担。您也同意众圈平台可以向您的电子邮件、手机号码等发送可能与本服务不相关的其他各类信息包括但限于商业广告等。您清楚知悉并同意，若您有任何事项需要通知众圈平台的，均应当按照本服务对外正式公布的联系方式或渠道系统进行提交，否则，众圈平台将无法收到您的通知。

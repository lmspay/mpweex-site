# 如何开发支付

支付涉及到商户开通支付，对应的小程序开通支付，小程序后端集成支付，小程序集成支付等环节。

具体步骤及文档：

  1. 开发者在小程序商户平台注册成为开发者。
  2. 在商户平台-账号中心，进行开发者认证及开通银商支付。
  3. 在商户平台-管理中心，创建小程序，等待审核通过。
  4. 在商户平台-管理中心-小程序管理，找到对应的小程序，在操作栏内找到开通支付按钮，点击即可开通对应小程序的支付功能。

  1. 小程序后端对接开放平台API，接入用户授权登录、支付相关功能。参见：开放平台API，聚合支付 Demo
  2. 小程序集成与开发者自己的后端对接，完成用户授权登录、用户资料获取等功能。参见：授权登录及信息安全，聚合支付 Demo
  3. 小程序集成支付。参见：授权登录及信息安全，聚合支付 Demo

# 小程序如何调试

小程序目前不支持WEB预览，因为WEB预览与真机预览差距较大，故开发者需要下载我们提供的playground进行调试。

下载地址：playground

安装完成后，先在电脑上进入到小程序开发目录，执行：

npm start

浏览器中会打开一个网页，注意不要关闭。在网页右上角有一个手机预览菜单，鼠标移上去会，会弹出二维码，使用下载的playground扫描即可进行预览调试。

TIP

手机和电脑要在同一网段哦

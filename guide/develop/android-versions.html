# 0.28.1

2020-05-08 发布0.28.1版本，版本编号2801

  • 更新小程序LOGO
  • 修改SDK页面加载流程，确定SDK初始化完成后，才允许页面加载
  • 修复缓存密钥加载失败BUG
  • 修改混淆参数
  • 添加小程序优惠券
  • 添加分享小程序到微信的功能
  • 修改开户逻辑，精简参数
  • 修改授权登录逻辑，从小程序平台获取用户信息并显示
  • 添加地区选择器
  • 修复推荐位点击可能点到空白导致无法跳转到小程序的BUG
  • 增加微信支付、支付宝支付、云闪付
  • 增加统一收银台
  • 修改useragent
  • 增加代金劵领取
  • 增加appVersionCode至环境变量
  • 修复标题栏标题过长，导致两侧按钮显示不全BUG
  • sys模块增加判断用户是否已登录API

# 0.28.0

2020-02-24 发布0.28.0版本

  • 同步weex版本至0.28.0
  • 增加arm64-v8a、x86CPU架构
  • 变更包名为com.lmspay.zq
  • SDK内部实现websocket
  • 修改picker模块，选择日期时支持指定格式
  • 修改volume模块，增加音量类型媒体、闹钟、来电
  • 增加跳转到指定页面
  • 增加原生View展示推荐位
  • 修复部分BUG

# 0.20.5

2019-12-24 发布0.20.5版本

  • 扩展webview组件，支持调用拍照及从相册选择
  • 修复导航栏push时指定状态栏字体前景色无效的BUG
  • 增加获取热门小程序的接口
  • 修复页面指定主题参数跟随BUG
  • 开放隐藏标题栏权限至普通小程序
  • 增加预留游客接口
  • 增加添加小程序、删除小程序toast提示

# 0.20.4

2019-12-18 发布0.20.4版本

  • 修复webview组件加载https内容时，无法同时加载http的图片的BUG
  • 增加用户登录状态同步接口
  • 修改UserInfoAdpater接口，增加Activity参数
  • 修改https请求封装，信任证书
  • 修改开户接口内部逻辑

# 0.20.3

2019-11-27 发布0.20.3版本

  • 修改webview组件，支持第三方APP跳转
  • 修改webview组件，支持定位

# 0.20.2

2019-11-19 发布0.20.2版本

  • 修改webview组件进度条
  • 修改webview组件，支持全屏播放视频
  • 修复actionSheet模块适配问题
  • 增加小程序需要登录才能访问的机制
  • 导航组件，增加显示/隐藏返回按钮API

# 0.20.1

2019-08-31发布0.20.1版本

  • 增加小程序源码加解密
  • 增加小程序安装包更新逻辑
  • 实现小程序缓存及离线访问
  • 增加小程序维护、冻结、下架等状态显示及相关逻辑
  • 增加光线及距离传感模块
  • 增加网络信息模块，用于判断当前网络类型
  • 增加游戏类小程序

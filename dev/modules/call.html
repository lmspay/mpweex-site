# call

用于拨打电话的模块

# API

# invoke

执行拨打电话操作

# invoke(number, callback)

  • @number {string}：电话号码
  • @callback {Function}：执行完该操作后的回调函数。回调函数将收到一个Response对象
    • result {string}：success仅表示执行操作成功，不表示电话拨打结果，error表示发生了错误
    • message {string}：错误描述，只当result为error时有此字段

Demo

var call = weex.requireModule('call');
const modal = weex.requireModule('modal');
call.invoke('10010', (e) => {
    if(e.result == 'success') {
        modal.toast({
            message: "called",
            gravity: "bottom",
            duration: 1
        });
    }
});
